Quick Start
Render the Survey

Use questions[] in the JSON.

Display statement and scale_labels (1–5).

Process Answers → Probabilities

Convert 1–5 to signals in -1, +1 (center = 3 = 0).

Apply direction (lower->... means lower = ↑ risk).

Drop negatives to 0 (only scenario-supporting signals contribute).

Aggregate using weights.

Apply:

Correlation dampening (correlation_examples).

Regime multiplier (regime_configs).

Time decay (decay_rates_annual).

Normalize with softmax τ=1.0 (normalize).

Compute Expected Asset Impacts

Singles: probability-weighted scenario shocks.

Overlaps (preferred)



impacts = (1 − W_pair) * singles + Σ w_ij * overlap_ij
where w_ij = 2 * p_i * p_j, W_pair ≤ 0.6


\


See integration.api_contract.expected_asset_impacts.

Map to Canonical Buckets

Use integration.asset_class_mapping.NEW_TO_CANONICAL.

HY Credit: add as new bucket.

Crypto: apply shocks to BTC (×1.0) and ETH (×1.25), cap ±80%.

IG Credit: use IG_CREDIT_SHOCKS.

Other buckets (Cash, Bills, Alternatives, Collectibles) have fallback formulas in OTHER_BUCKET_RULES.

🧪 Regimes & Uncertainty

Decay rates: exponential per-scenario (annualized).

Monte Carlo: run 1000 iterations with regime-scaled noise to get CI bands (95% intervals).
