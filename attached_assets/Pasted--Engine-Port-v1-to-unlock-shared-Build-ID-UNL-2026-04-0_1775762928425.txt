## Rules

- Do NOT modify any existing files in this project
- Only create files in `_export/`
- Every output file must be self-contained — no imports from
  this project's src/ directories
- Output files may import from each other within `_export/`
  (e.g. simulationEngine.ts may import from simulationConfig.ts)
- No Express, no Drizzle, no OpenAI, no HTTP in any output file
- All functions must be pure (input → output, no side effects,
  no global state mutation)
- TypeScript strict mode — no `any` types unless absolutely
  unavoidable, and document why
