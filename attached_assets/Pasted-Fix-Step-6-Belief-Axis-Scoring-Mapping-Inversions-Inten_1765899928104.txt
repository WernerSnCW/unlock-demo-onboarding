4) Fix intensity labels to be deterministic and consistent

Intensity is based on absolute value of the axis score:

|score| < 0.20 → Neutral

0.20 ≤ |score| < 0.50 → Light

0.50 ≤ |score| < 0.80 → Moderate

|score| ≥ 0.80 → Strong

Direction rules:

score > +0.20 → TOWARDS

score < −0.20 → AWAY

otherwise → NEUTRAL

Important: Do not “dampen” a +1.00 into +0.50. If a user selects Strongly Agree, the score must remain ±1.00 (except volatility inversion).
