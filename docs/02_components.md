# Component Library

## NavBar
- Displays logo and navigation links.
- Sticky top bar, responsive.
- Supports light/dark themes.

## Card
- Surface container with shadow.
- Used for grouping content blocks.
- Respects `--color-surface` and `--shadow-md`.

## TrustMarker
Status badges shown next to businesses:
- ✅ Verified (data confidence check passed)
- 🔒 Premium (feature requires paid plan)
- 📊 Peer % (aggregated peer sentiment metric)

## Accordion
- Expand/collapse for FAQs, Q&A, or reports.
- Smooth transitions.
- Only one section open at a time.

## ExportModal
- Allows exporting a snapshot report.
- Options: PDF / CSV / JSON.
- Shows disclaimer: “No advice; businesses only.”

## ChartStub
- Placeholder for charts (bar, line, pie).
- Uses demo data until wired into APIs.
